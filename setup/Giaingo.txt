> SQL SERVER MANAGEMENT STUDIO *********************************
- LOGIN VỚI QUYỀN WINDOWS AUTHENTICATION
* SETTING USER SA ************
- CHỌN SECURITY | LOGINS | SA | NHÁY CHUỘT TRÁI 2 LẦN SA
- CHỌN MỤC STATUS, TÍCH VÀO (o) GRAND VÀ (o) ENABLE ĐỂ CHO PHÉP LOGIN VÀO SQL SERVER BẰNG USER SA (SYSTEM ADMIN)
- CHỌN MỤC GENERALS ĐỂ ĐỔI PASSWORD, RETYPE
- OK TẮT MÀN HÌNH

* SETTING CÁC KẾT NỐI VÀO SQL SERVER
- CÓ 2 CÁCH KẾT NỐI VÀO SQL SERVER
     - DÙNG QUYỀN WINDOWS ĐỂ VÀO - WINDOWS AUTHENTICATION, MÌNH ĐANG DÙNG 
     - DÙNG MIXED MODE, WINDOWS AUTHEN VÀ USER SA CẢ 2 ĐỀU ĐƯỢC
- CẤU HÌNH MIX MODE NHƯ SAU: 
- PHẢI CHUỘT VÀO CONNECTION TRÊN MÀN HÌNH | PROPERTIES
  CHỌN MỤC SECURITY | TÍCH OPTION (o) SQL Server and Window Authen mode
- PHẢI CHUỘT TRÊN CONNECTION, CHỌN RESTART SERVER (KO PHẢI CHỌN DISCONNECT, CONNECT LẠI)
- ĐỂ SERVER GHI NHẬN NHỮNG THAY ĐỔI VỀ USER SA VÀ CÁCH CONNECT TỚI NÓ

* TEST THỬ CONNECT
- DISCONECT KẾT NỐI = NHẤN NÚT RÚT Ổ CẮM
- KẾT NỐI LẠI BẰNG NÚT PHÍCH CẮM, NHẤN CHỌN SQL SERVER AUNTHENTICATION VÀ NHẬP USER sa, PASSWORD VỪA ĐỔI, TÍCH CHỌN [X]
TRUST SERVER CERFIFICATE 
CONNECT THÀNH CÔNG


>>>>>>>>> SQL SERVER CONFIGURATION MANAGER >>>>>>>>>>>>>>>

- ĐÂY LÀ APP GIÚP TA SETUP PORT, CHO PHÉP TA TRUY XUẤT VÀO SQL SERVER TỪ CODE, TỪ APP KHÁC!!!!!!!
- CẤU HÌNH GIAO THỨC TCP/IP (TRUYỀN DỮ LIỆU QUA MẠNG)
- CẤU HÌNH CỔNG LẮNG NGHE: ĐẶT MÃ SỐ CHO APP (0...06555365 = 2^16"
1433 LÀ MẶC ĐỊNH DÀNH CHO SQL SERVER

* CÓ NHIỀU CÁCH SỜ APP NÀY
- GÕ CONFI Ở Ô SERARCH KẾ BÊN BIỂU TƯỢNG WINDOWS Ở THANH TASKBAR
- CHỌNG TÊN APP ĐỀ XUẤT: SQL SERVER XXX CONFIGUARATION MANAGER

- NẾU KO XỔ, THÌ PHẢI CHUẬT TRÊN BIỂU TƯỢNG WINDOWS, CHỌN RUN, GÕ SQLServerManager16.msc (11, 12, 13, 14, 15, 16)

- CÁCH 3: MẤT CÔNG HƠN, TÌM XEM NÓ Ở ĐÂU TRÊN Ổ ĐĨA
C:\Windows\SysWOW64\SQLServerManager1X.msc
